#Taller Práctico de NumPy

Este repositorio contiene el desarrollo de un taller práctico utilizando la librería **NumPy** en Python, enfocado en el manejo de arrays y operaciones matemáticas fundamentales.

---

## 📁 Contenido

El repositorio incluye notebooks desarrollados en Google Colab:

* 📓 `Ejercicios_AlanEspaña.ipynb`
  Contiene ejercicios relacionados con operaciones básicas de NumPy (arrays, operaciones matemáticas, etc.)

* 📓 `Modulos_NumPy_AlanEspaña.ipynb`
  Incluye ejercicios más avanzados o aplicaciones prácticas (transformaciones, análisis o visualización)

---

## Objetivo

Aplicar conceptos fundamentales de NumPy como:

* Creación y manipulación de arrays
* Operaciones matemáticas vectorizadas
* Uso eficiente de estructuras numéricas
* Resolución de problemas prácticos

---

## Cómo usar

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/alanmartin-dev/Taller_Practica_NumPy.git
   ```

2. Abrir los notebooks en:

   * Google Colab
   * Jupyter Notebook
